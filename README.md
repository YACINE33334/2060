# متجر إلكتروني خفيف

متجر بسيط للاستخدام الشخصي مع إدارة منتجات وتكامل Facebook Pixel.

## التشغيل

```bash
npm install
npm start
```

ثم افتح: http://localhost:3000

## المميزات

- **إضافة منتجات** من لوحة الإدارة `/admin`
- **Facebook Pixel** — أدخل معرف البكسل في الإعدادات
- **واتساب** — زر طلب يفتح محادثة واتساب
- **قاعدة بيانات SQLite** — خفيفة، ملف واحد بدون إعداد
- **واجهة عربية** كاملة

## الإعدادات

في `config.json`:

- `storeName`: اسم المتجر
- `adminPassword`: كلمة مرور لوحة الإدارة (غيّرها!)
- `facebookPixelId`: معرف Facebook Pixel (يمكن تعيينه من لوحة الإدارة أيضاً)

## هيكل المشروع

```
STORE/
├── server.js      # الخادم الرئيسي
├── db.js          # قاعدة البيانات
├── config.json    # إعدادات
├── views/         # قوالب الصفحات
├── public/        # ملفات ثابتة
└── data/          # قاعدة SQLite (تُنشأ تلقائياً)
```
